Introduction

In the ever-evoluting web environment, Dokeos has to rely on state-of-the-art third-party software in order to provide you with useful and practical features. Some of this third-party software usually come bundled with your operating system, but some don't. This means that if that software is not installed on your Dokeos server, along with Dokeos, you might not enjoy the best possible experience with your Dokeos platform. To avoid this problem, most of the sofware we used is bundled into a typical Dokeos install.
However, if such a third-party software is too big to be included, or if it doesn't make sense to includ it for whatever reason, you must make sure it is installed on your system. This is the goal of this list: to help you keep a track of what software is needed and, as much as possible, where to get the required information to install it.

The third-party software we have bundled inside Dokeos can be found in the Included Dependencies section below. The software you need to install manually is located in the section.

Dependencies

  • Apache 1.3 or 2
  • PHP5.0.0+ with MySQL bindings. Progressive upload bar only works with PHP5.2.0+, although PHP4.3.0 will work for most things.
  • MySQL database server

Included Dependencies

  • FormValidator
  • kSes
  • MIMETex
  • MP3Player
  • PCLZip
  • PHPMailer
  • xajax
  • PEAR::HTML_Common
  • PEAR::HTML_QuickForm
